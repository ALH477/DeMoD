# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1.

In short: be respectful. Harassment, discrimination, and personal attacks are not
welcome; technical disagreement is. Maintainers may remove contributions and
contributors that violate these norms.

Report unacceptable behavior to **alh477@proton.me**. Reports are handled
confidentially.

Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
