# Security Policy

## Supported versions

This project is pre-1.0 and under active development; security fixes land on
`main`. Pin a commit if you need stability.

## Reporting a vulnerability

Please **do not** open a public issue for security problems. Email
**alh477@demod.ltd** with a description, reproduction steps, and impact. You'll get
an acknowledgement, and we'll coordinate a fix and disclosure timeline with you.

Scope note: this repository is the framework + audio stack. Product-level concerns
(signing keys, entitlements, marketplace) live in separate, non-public repositories.
