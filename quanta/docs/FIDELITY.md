# Fidelity — objective + perceptual results

Regenerated by `test/run.sh` (gates F, P) and `tools/perceptual.py`. Two lenses:

- **LSD** (`tools/metrics.py`) — magnitude log-spectral distance, DC-blocker
  compensated, floored at −80 dB re frame peak. Every bin weighted equally.
- **NMR** (`tools/perceptual.py`) — Noise-to-Mask Ratio: masking-weighted
  *magnitude-envelope* distortion. `< 0 dB` ⇒ error below the masking threshold
  (inaudible). Magnitude-domain on purpose: Quanta's residual is a **noise-
  substitution** model (reproduces the spectral envelope with a *different* random
  realization), so waveform-difference metrics — including ViSQOL/PESQ — wrongly
  punish perceptually-inaudible phase decorrelation. NMR is a documented objective
  **proxy**, not a calibrated MOS; an absolute "transparent at N kbps" claim still
  needs a blinded listening study (ROADMAP §2.3).

## Results (v0.2.x, `--mode near` where streamed)

| Corpus | Rate | LSD on | LSD atoms-only | NMR on (median) | NMR atoms-only | % audible frames (on) |
|---|---|---|---|---|---|---|
| tonal (synthetic) | — | 1.64 dB | 1.55 dB | **−30.6 dB** | −30.8 dB | 0.4% |
| hybrid `src` (synthetic) | — | 10.2 dB | 15.9 dB | +5.0 dB | +3.5 dB | 84% |
| **Bach Aria (real, CC0)** | 94.9 kbps | 8.66 dB | 6.72 dB | **−0.97 dB** | −0.88 dB | 37% |

## Reading these honestly

- **Tonal content is perceptually transparent** (NMR −30 dB, ~0% audible): the
  Phase-1 tonality-aware residual no longer fills spectral valleys with audible
  noise. Gate F (LSD) + Gate P (NMR) both hold.
- **LSD and NMR disagree on the residual**, and NMR is right about audibility: on the
  Bach Aria, LSD says the residual *hurts* (8.66 vs 6.72 dB) while NMR says it is
  perceptually *neutral* (−0.97 vs −0.88 dB) — the added valley energy is masked.
- **Not yet transparent on real piano at ~95 kbps**: ~37% of frames carry an audible
  NMR error. This is the honest state and the target for further work (bit
  allocation, more atoms/rate, ROADMAP H2/§2.3), not a claim of transparency.

## Hi-res transparency: the noise-substitution ceiling (Track B)

Measured on genuine 96 kHz / 24-bit CC0 masters (Open Goldberg Aria + Variatio 5),
lossy transparent preset (`--quality 10`), `make transparency`:

| clip | src-SNR | LSD | NMR median | audible frames |
|---|---|---|---|---|
| Aria 96 k | +16.6 dB | 6.64 dB | −7.08 dB | 8.5% |
| Aria 48 k | +14.3 dB | 6.85 dB | −8.95 dB | 2.5% |
| Variatio 5 96 k (dense) | +32.6 dB | 8.77 dB | −9.63 dB | 0.0% |

**The ceiling is the residual model, not the atom budget.** Raising the atom budget
(k 16k→65k), the pursuit-stop SNR (60→100 dB), or lowering the `--floor` salience knob
does **not** move source-SNR (stayed +14.34 → +14.36): matching pursuit *saturates* once
the coherent partials are captured, and Gabor atoms need O(N) of them to represent the
broadband-noise remainder. So the lossy tier is **perceptually** transparent-leaning
(NMR masked) but **not** waveform-transparent — the two are decoupled.

## Bit-transparent tier — coherent residual (`--coherent`, spec §5.3)

Storing the true post-atom residual breaks the ceiling. Null-vs-**source** goes from
−3 dBFS (lossy) to below the source's own LSB, on a dialable `--cbits` knob:

| tier | null vs source | coded (zstd) | note |
|---|---|---|---|
| lossy (atoms + noise) | −3 dBFS | ~250 kbps | perceptually transparent, not bit-exact |
| `--cbits 8` | −66 dBFS | ~167 kbps | |
| `--cbits 12` | **−90 dBFS** | ~307 kbps | inaudible; **beats FLAC** (527 kbps) |
| `--cbits 16` | **−114 dBFS** | ~632 kbps | below 16-bit source LSB; FLAC-class |

96 k/24 stereo at 16-bit: null −109.6 dBFS, ~2354 kbps zstd vs 4608 kbps PCM. The frozen
static `.dsp` nulls the C render to −280 dBFS (determinism) **and** the source to −114 dBFS
— a bit-transparent decoder that is a pure Faust program. **Honest:** not a compression win
over FLAC (the residual is high-entropy); the win is one scalable file (lossy ↔ bit-exact)
and the decoder-as-`.dsp`. Gate: `make coherent`.

## Analytic resynthesis vs phase vocoder (`quanta-score`, spec §5.4)

Because the score is an explicit sinusoidal-grain + band-residual model, studio transforms
are *edits of the model*, not time-frequency re-estimation — so several artifacts a phase
vocoder must fight are absent **by construction**:

- **Time-stretch** scales each grain's `onset` and `dur`; pitch is invariant because grain
  phase is `freq·tl/sr` (no frequency-bin re-estimation, so no transient smearing / "phasiness").
  `--keep-transients` holds `layer==1` grain length, keeping attacks razor-sharp at any factor.
- **Pitch shift** moves partials directly; the `--formant` family re-weights grain amplitude by a
  spectral envelope `E(f)` so formant peaks stay put while partials move (naive transpose drags
  them). Two modes, and we **measured** rather than assumed which to default to: on a real 96 k
  harpsichord master (+7 st), per-window (85 ms) spectral-centroid deviation from the reference was
  **naive 0.188, global `--formant` 0.124, per-frame `--formant-dyn` 0.134** — i.e. per-frame is
  *worse* on stationary-timbre music (its frame-to-frame variance buys nothing when the envelope
  isn't moving), so **`--formant` (one global envelope) is the default**. `--formant-dyn` (per-frame,
  ~85 ms, shrunk toward the global prior) is retained for material with genuinely time-varying
  formants (voices, evolving mixes), where the locality has something to track.
- **EQ / width / gain** are exact scalar edits on atom amplitudes and the 24 residual-band gains.

Every transform still **freezes to a `.dsp` that nulls its own render ≤ −120 dBFS** (Gate I) —
determinism survives editing. Honest limits: `--formant` (global) is the default because it
measured tighter than per-frame on stationary-timbre music (`--formant-dyn` is for moving
formants), `pitch` can't transpose the fixed-band noise residual, and large stretch
has mild non-constant-energy overlap ripple. **Editing forfeits bit-transparency by
construction**: changing the signal invalidates a stored `source − atoms` coherent residual, so
`quanta-score` drops the coherent layer (§5.3) and the edited master is the analytic
(atoms + noise) tier — expected, not a regression (the point of an edit is to change the sound).
