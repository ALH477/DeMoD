# Fidelity — objective + perceptual results

Regenerated by `test/run.sh` (gates F, P) and `tools/perceptual.py`. Two lenses:

- **LSD** (`tools/metrics.py`) — magnitude log-spectral distance, DC-blocker
  compensated, floored at −80 dB re frame peak. Every bin weighted equally.
- **NMR** (`tools/perceptual.py`) — Noise-to-Mask Ratio: masking-weighted
  *magnitude-envelope* distortion. `< 0 dB` ⇒ error below the masking threshold
  (inaudible). Magnitude-domain on purpose: Quanta's residual is a **noise-
  substitution** model (reproduces the spectral envelope with a *different* random
  realization), so waveform-difference metrics — including ViSQOL/PESQ — wrongly
  punish perceptually-inaudible phase decorrelation. NMR is a documented objective
  **proxy**, not a calibrated MOS; an absolute "transparent at N kbps" claim still
  needs a blinded listening study (ROADMAP §2.3).

## Reading these honestly

- **Tonal content is perceptually transparent** (NMR −30 dB, ~0% audible): the
  Phase-1 tonality-aware residual no longer fills spectral valleys with audible
  noise. Gate F (LSD) + Gate P (NMR) both hold.
- **LSD and NMR disagree on the residual**, and NMR is right about audibility: on the
  Bach Aria, LSD says the residual *hurts* (8.66 vs 6.72 dB) while NMR says it is
  perceptually *neutral* (−0.97 vs −0.88 dB) — the added valley energy is masked.
- **Not yet transparent on real piano at ~95 kbps**: ~37% of frames carry an audible
  NMR error. This is the honest state and the target for further work (bit
  allocation, more atoms/rate, ROADMAP H2/§2.3), not a claim of transparency.
